# Sticky Notes — Infinite Canvas

A minimal sticky notes app with an infinite canvas. Create, edit, move, resize, and delete notes — all synced to your account in real time.

Built with React, TypeScript, Tailwind CSS, and Lovable Cloud. All done with lovable!

Link to site: https://infinitenotes.lovable.app/
