# infiniteNotes — Infinite Canvas

A minimal sticky notes app with an infinite canvas. Create, edit, move and resize notes — all synced to your account in real time.

Built with React, TypeScript, Tailwind CSS, and Lovable Cloud. All done with Lovable!

Link to site: https://infinitenotes.lovable.app/
